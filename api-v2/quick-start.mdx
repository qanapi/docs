## 🚀 Officially supported Qanapi API v2 SDKs

### 1️⃣ Install the SDK

**TypeScript / JavaScript:**
(Node.js, Bun, Deno, etc.)

```bash
npm install qanapi
```

**Python:**

```bash
pip install qanapi
```

**Golang:**

```bash
go get -u 'github.com/qanapi/qanapi-sdk-golang'
```
